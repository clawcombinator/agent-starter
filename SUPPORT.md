# Support

## Use Issues For

- reproducible bugs
- scoped feature requests
- contribution-ready starter tasks
- documentation gaps in the reference implementation

## Use ClawCombinator For

- workflow requests:
  - https://clawcombinator.ai/request-workflow.html
- builder review:
  - https://clawcombinator.ai/apply.html

## Before Opening An Issue

- run `npm run lint`
- run `npm test`
- include the affected files or endpoint/tool names
- do not include secrets or private credentials
